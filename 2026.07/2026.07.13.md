## 2. 프로그래머스 문제 풀이

### [4단 고음](https://school.programmers.co.kr/learn/courses/30/lessons/1831)

```cpp
#include <vector>
#include <string>
#include <cmath>

void DFS(const int n, int zero, int& answer){
    if(n == 3) {
        if(zero == 2) answer++;
        return;
    }
    if(n % 3 == 0 && zero >= 2) DFS(n/3, zero-2, answer);
    if(n > 3 && (log(n)/log(3))*2 > zero) DFS(n-1, zero+1, answer);
}

// 전역 변수를 정의할 경우 함수 내에 초기화 코드를 꼭 작성해주세요.
int solution(int n) {
    int answer = 0;
    DFS(n-2, 2, answer);
    return answer;
}
```

* **역방향 백트래킹** 문제
* 문자열의 특징은 가장 앞 문자는 항상 **\***, 가장 뒤 두 문자는 항상 **\+**
* 따라서 시작할 때 2를 빼주고, `n`이 3인 경우 `answer`를 1 증가
* **\*** 을 쓰기 위해서는 뒤에 **반드시 \+의 개수가 2개 이상** 있어야 함
* 같은 이유로 **\+** 를 쓰기 위해서는 지금까지 저장된 **\+** 를 지울 수 있는 만큼의 `n`이 존재해야 함
