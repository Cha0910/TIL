## 2. 프로그래머스 문제 풀이

### [프로세스](https://school.programmers.co.kr/learn/courses/30/lessons/42587)

```cpp
#include <string>
#include <vector>
#include <queue>

using namespace std;

int solution(vector<int> priorities, int location) {
    int answer = 0, cur = 0;
    priority_queue<int> pq;
    for(int i=0; i<priorities.size(); i++) pq.push(priorities[i]);
    
    while(!pq.empty()){
        if(pq.top() == priorities[cur]){
            pq.pop();
            answer++;
            if(cur == location) break;
        }
        cur++;
        cur %= priorities.size();
    }
    
    return answer;
}
```
