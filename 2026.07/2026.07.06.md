### [124 나라의 숫자](https://school.programmers.co.kr/learn/courses/30/lessons/12899)

```cpp
#include <string>
#include <vector>
#include <algorithm>

using namespace std;

string solution(int n) {
    string answer = "";
    while(n>0){
        if(n%3 != 0) answer += (n % 3) + '0';
        else {
            answer += 4 + '0';
            n--;
        }
        n /= 3;
    }
    reverse(answer.begin(), answer.end());
    return answer;
}
```

* **진법 변환** 문제
* 사실상 `0`을 `4`로 사용하는 **3진법 변환** 문제
* 따라서 나머지가 0인 경우, `4`를 붙이면 됨
* 기존의 `3`진법에서 `0`은 단순 **자릿수를 늘려주는 수**이지만 이 문제에서의 `4`는 단순 자릿수가 아닌, **3의 값을 갖고 있는 수 **이므로 몫을 `1` 빼주어 다음 자릿수의 수를 감소시켜야 함

### [[3차] n진수 게임](https://school.programmers.co.kr/learn/courses/30/lessons/17687)

```cpp
#include <string>
#include <vector>
#include <algorithm>

using namespace std;

string convert(int t, int n){
    string result = "";
    while(t>0){
        int data = t % n;
        if(data >= 10) result += (data-10 + 'A');
        else result += data + '0';
        t /= n;
    }
    reverse(result.begin(), result.end());
    return result;
}

string solution(int n, int t, int m, int p) {
    string answer = "";
    string list = "0";
    int i = 0;
    while(list.length() < t*m) list += convert(++i, n);
    for(int i=0; i<t; i++) answer += list[(i * m) + (p - 1)];
    
    return answer;
}
```

* **진법 변환** 문제
* 게임 진행 중 나오는 모든 수를 `list`에 저장
* 이후 인덱스 접근으로 **튜브의 차례에 말해야 하는 숫자**를 `answer`에 저장

### [다음 큰 숫자](https://school.programmers.co.kr/learn/courses/30/lessons/12911)

```cpp
#include <string>
#include <vector>

using namespace std;

int solution(int n) {
    int answer, count = -2;
    for(int i=0; i<20; i++){
        if((n >> i & 1) == 1){
            answer = n + (1 << i);
            break;
        }
    }
    
    int x = n ^ answer;
    for(int i=0; i<20; i++)
        if((1 & (x >> i)) == 1) count++;
    answer |= (1 << count) - 1;
    
    return answer;
}
```

* **비트 연산** 문제
* 첫 번째 반복문에서 **가장 오른쪽에 위치한** `1`을 찾음
* 그 위치에 1을 더하면 연속한 1 다음에 위치한 0이 1로 바뀌면서 기존의 1이 전부 0으로 바뀜
* 이후 기존의 `n`과 `answer`를 **XOR** 연산으로 **변경된 숫자의 개수**를 셈
* 가장 오른쪽에서부터 **변경된 숫자의 개수 -2**개의 `0`을 `1`로 바꿔주면 정답
