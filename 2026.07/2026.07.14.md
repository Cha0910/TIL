## 2. 프로그래머스 문제 풀이

### [타겟 넘버](https://school.programmers.co.kr/learn/courses/30/lessons/43165)

```cpp
#include <string>
#include <vector>

using namespace std;

int DFS(const vector<int>& numbers, const int& target, int sum, int index){
    if(index == numbers.size()){
        if(sum == target) return 1;
        else return 0;
    }
    int result = 0;
    result += DFS(numbers, target, sum + numbers[index], index + 1);
    result += DFS(numbers, target, sum - numbers[index], index + 1);
    return result;
}

int solution(vector<int> numbers, int target) {
    int answer = 0;
    answer += DFS(numbers, target, 0, 0);
    return answer;
}
```

* **DFS** 문제
* `+`, `-`의 모든 경우를 계산해보면 됨
* `numbers`의 길이가 최대 20이라 모든 경우의 수는 `2^20`

### [[1차] 비밀지도](https://school.programmers.co.kr/learn/courses/30/lessons/17681)

```cpp
#include <string>
#include <vector>
#include <algorithm>
using namespace std;
string parse(int n){
    string result = "";
    while(n>0){
        result += n%2 == 1 ? "#" : " ";
        n /= 2;
    }
    return result;
}

vector<string> solution(int n, vector<int> arr1, vector<int> arr2) {
    vector<string> answer;
    int len = 0;

    for(int i=0; i<arr1.size(); i++){
        answer.push_back(parse(arr1[i] | arr2[i]));
        len = max(len, (int)answer.back().length());
    }
    for(int i=0; i<answer.size(); i++){
        answer[i] += string(len - answer[i].length(), ' ');
        reverse(answer[i].begin(), answer[i].end());
    }
        
    return answer;
}
```

* **비트 연산**, **문자열** 문제
* 두 지도를 겹쳐서 해독하므로 **OR 연산**
