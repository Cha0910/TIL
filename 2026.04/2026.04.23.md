## 2. RTTI

* **Run Time Type Information** 의 약자로 프로그램 **실행 중** 개체의 타입이 결정될 수 있도록 하는 **메커니즘**

* 주로 부모 타입의 포인터, 참조가 자식 타입의 인스턴스를 가리키는 경우 이를 안전하게 **다운 캐스팅** 하는데 사용

### vTable

* 다형성을 활용하기 위해 만드는 **가상 함수 테이블**

* **C++** 에서 클래스의 멤버 함수 중 하나라도 `virtual` 키워드를 붙인 함수가 존재한다면, **vTable**을 생성하며 **vPtr** 이라는 포인터가 이를 가리킴

* 각각의 **vTable**은 해당 클래스의 가상 함수들의 **주소**를 저장하고 있음

* **업 캐스팅**된 포인터에서 가상 함수를 실행할 때, **vTable**의 주소 값을 확인하여 **업 캐스팅**된 타입에 **알맞은 가상 함수**를 실행함

### dynamic_cast

* **C++** 에서 주로 안전하게 **다운 캐스팅**할 때 사용하는 **연산자**

* `dynamic_cast`를 사용하기 위해서는 각 클래스가 적어도 **하나 이상의 가상 함수**를 포함하고 있어야 함 (vTable을 필요로 하기 때문)

* 런타임 도중 안정성을 보장하기에 **static_cast** 보다 약간의 **오버헤드**가 존재

* **포인터**의 경우, 캐스팅 성공 시 `캐스팅된 포인터`를 반환하며, 실패 시 `nullptr`을 반환함

* **참조**의 경우, 캐스팅 성공 시 `캐스팅된 참조 타입`을 반환하며, 실패 시 `std::bad_cast`을 예외를 발생시킴

* **예제 코드**
    ```cpp
    #include <iostream>

    class Base {
    public:
        virtual ~Base() {}  // vTable 생성을 위한 가상 소멸자
    };

    class Derived : public Base {
        void print() {
            std::cout << "다운 캐스팅 성공" << std::endl;
        }
    };

    int main() {
        // 자식 타입의 인스턴스를 갖는 부모 타입의 포인터 (업 캐스팅)
        Base* upCasting = new Derived();    

        // upCasting->print(); // 컴파일 에러, 부모 클래스는 자식의 함수에 대한 데이터가 없음

        // 업 캐스팅 된 부모 타입의 포인터를 자식 타입의 포인터로 캐스팅 (다운 캐스팅)
        Derived* downCasting = dynamic_cast<Derived*>(upCasting);   

        
        if(downCasting){           // 캐스팅 성공 여부 확인
            downCasting->print();  // Output: 다운 캐스팅 성공
        }

        return 0;
    }
    ```
### type_info 

* 프로그램이 **실행 중**일 때, 타입의 정보를 표현하는 **클래스**

* 사용하기 위해서는 `<typeinfo>` 헤더를 포함해야 함

* **복사** 및 **대입**이 **불가능** 함

* `==`, `!=` 연산자를 활용하여 **비교 연산**이 가능

* `name()` 함수를 통하여 타입의 이름을 얻을 수 있지만, 컴파일러별로 반환하는 형식이 다르므로 비교 연산자를 사용하는 것을 권장함

* **인터페이스**는 아래와 같음

    ```cpp
    class type_info {
    public:
        type_info(const type_info& rhs) = delete; // cannot be copied
        virtual ~type_info();
        size_t hash_code() const;
        _CRTIMP_PURE bool operator==(const type_info& rhs) const;
        type_info& operator=(const type_info& rhs) = delete; // cannot be copied
        _CRTIMP_PURE bool operator!=(const type_info& rhs) const;
        _CRTIMP_PURE int before(const type_info& rhs) const;
        size_t hash_code() const noexcept;
        _CRTIMP_PURE const char* name() const;
        _CRTIMP_PURE const char* raw_name() const;
    };
    ```

### type_id

* **타입**이나 **인스턴스**를 인자로 받아 인자의 `type_info`를 반환하는 **연산자**

* 가상 함수가 없는 기본 타입이나, 클래스를 인자로 받는 경우 **정적 바인딩**을 하며, 가상 함수가 하나라도 포함된 클래스를 인자로 받는 경우 **동적 바인딩을** 실시함

* 캐스팅하기 전, 인스턴스의 **타입을 확인**할 수 있어 안전하게 캐스팅이 가능함

* `typeid`의 인자로 **nullptr**을 줄 경우, `std::bad_typeid` 예외를 발생시킴

* **예제 코드**
    ```cpp
    #include <iostream>
    #include <typeinfo>

    class Base {
    public:
        virtual ~Base() {}  // vTable 생성을 위한 가상 소멸자
    };

    class Derived : public Base {
    public:
        void print() {
            std::cout << "다운 캐스팅 성공" << std::endl;
        }
    };

    int main() {
        Base* upCasting = new Derived();

        // upCasting-> print(); // 컴파일 에러

        // upCasting의 타입 자체는 Base*임
        std::cout << typeid(upCasting).name() << std::endl;     // Output: class Base *

        // 역참조 후 인자로 넣어야 포인터가 가리키는 타입을 알 수 있음
        std::cout << typeid(*upCasting).name() << std::endl;    // Output: class Derived
        
        if(typeid(*upCasting) == typeid(Derived)){  // typeid를 사용하여 타입을 확인

            // 타입을 이미 확인 했으므로, static_cast를 사용해도 안전함
            Derived* downCasting = static_cast<Derived*>(upCasting);   

            if(downCasting){           // 캐스팅 성공 여부 확인
                downCasting->print();  // Output: 다운 캐스팅 성공
            }
        }

        return 0;
    }
    ```

## 3. RAII

* **Resource acquisition is initialization** 의 약자로 자원의 **해제**를 객체의 **생명 주기**에 위임하는 **메커니즘**

* 전통적인 **C++** 에서 동적 할당 할때 `new`, `delete`를 사용하여 개발자가 직접 자원을 관리함

* **RAII**의 핵심은 사용자가 자원을 직접 관리하는 것이 아닌 **모든 자원 관리를 객체가 전담**하게 하는 것

* **STL**에서 제공하는 `스마트 포인터`, `mutex`, `vector,` `fstream` 등은 **RAII**가 적용된 것

* **코드 예제**
    ```cpp
    #include <iostream>

    class Array {
    private:
        int* data;

    public:
        // 생성자에서 동적 할당
        Array(const int size){
            data = new int[size];
        }

        // 소멸자에서 해제
        ~Array(){
            delete[] data;
        }
    };


    int main() {
        Array arr(10);  // 따로 해제하지 않아도 해당 함수 블록에서 탈출하면, 자동으로 해제 됨

        return 0;
    }
    ```
