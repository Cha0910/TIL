### [n^2 배열 자르기](https://school.programmers.co.kr/learn/courses/30/lessons/87390)

```cpp
#include <string>
#include <vector>

using namespace std;

vector<int> solution(int n, long long left, long long right) {
    vector<int> answer(right-left+1);
    for(long long i=left; i<=right; i++)
        answer[i - left] = (max(i/n, i%n) + 1);
    return answer;
}
```

* **구현**? 문제
* `n`의 범위가 `1 ≤ n ≤ 10^7`이라 배열을 만들고, 직접 세면 시간, 공간 복잡도가 너무 큼
* 배열의 특성을 이용해 일자로 펼쳐진 위치를 2차원 배열의 위치로 바꿔서 계산

### [억억단을 외우자](https://school.programmers.co.kr/learn/courses/30/lessons/138475)

```cpp
#include <string>
#include <vector>
#include <cmath>
#include <set>
#include <unordered_map>
#include <algorithm>
using namespace std;

// 소수 판별
bool isPrime(int n){
    if (n <= 1) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false; 
    
    for(int i=3; i*i<=n; i++)
        if(n % i == 0) return false;
    return true;
}

// 소인수 분해
unordered_map<int, int> primeFactorization(const set<int>& prime, int n){
    unordered_map<int, int> result;
    for(const int& p : prime){
        while(n%p == 0){
            result[p]++;
            n /= p;
        }
        if(n == 1 || p*p > n) break;   
    }
    if (n > 1)  result[n]++;
    
    return result;
}

vector<int> solution(int e, vector<int> starts) {
    vector<int> answer(starts.size());
    set<int> prime;
    vector<long long> count(e+1);
    for(int i=2; i<=e; i++)
        if(isPrime(i)) prime.insert(i);

    for(int i=1; i<=e; i++){
        unordered_map<int, int> pf = primeFactorization(prime, i);
        long long temp = 1;
        for(const auto& [p, c] : pf)
            temp *= (c+1);
        count[i] = temp;
    }
    
    int maxCount = 0;
    int max;
    vector<int> slice(e+1, 0);
    for(int i=e; i>=1; i--){
        if(count[i] >= maxCount) {
            maxCount = count[i];
            max = i;
        }
        slice[i] = max;
    }
    
    for(int i=0; i<starts.size(); i++)
        answer[i] = slice[starts[i]];

    return answer;
}
```

* **수학**, **구현** 문제
* 문제에서 요구하는 정답은 어떤 수 `N`을 소인수분해 한 후 각 소인수의 지수 + 1을 모두 곱한 것
* 입력받은 `e`보다 작은 소수를 모두 구한 뒤 `1~e`까지의 수를 모두 소인수분해 해 `count`에 저장
* 이후 각 구간별 최댓값을 저장한 후 `starts`의 값에 따라 `answer`에 저장
* 다 풀고 나서 다른 사람들 코드를 보니 **에라토스테네스의 체** 라는게 있었음

### [등산코스 정하기](https://school.programmers.co.kr/learn/courses/30/lessons/118669)

```cpp
#include <string>
#include <vector>
#include <unordered_set>
#include <algorithm>
#include <queue>
using namespace std;

// 다익스트라
vector<int> dijkstra(const vector<vector<pair<int, int>>>& graph, 
                     const unordered_set<int>& gate, const unordered_set<int>& summit){
    vector<int> result(graph.size(), 10000001);
    priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> pq;
    
    // 모든 출입구를 우선순위큐에 넣음
    for(const int g : gate){
        pq.push({0, g});
        result[g] = 0;
    }
    
    // 다익스트라 실행
    while(!pq.empty()){
        int max = pq.top().first;
        int current = pq.top().second;
        pq.pop();
        
        for(const pair<int, int>& edges : graph[current]){
            int weight = edges.first;
            int next = edges.second;
            weight = max > weight ? max : weight;
            if(result[next] > weight) {
                result[next] = weight;
                if(gate.count(next) == 0 && summit.count(next) == 0) 
                    pq.push({weight, next});
            }
        }
    }
    
    return result;
}

vector<int> solution(int n, vector<vector<int>> paths, vector<int> gates, vector<int> summits) {
    vector<int> answer = {-1, 10000001};
    vector<vector<pair<int, int>>> graph(n+1);  // 비용, 목적지
    unordered_set<int> gate, summit;            // 출입구, 산봉우리를 저장하는 set
    sort(summits.begin(), summits.end());       // 출입구를 정렬
    
    // 그래프 생성
    for(const vector<int>& path : paths){
        graph[path[0]].push_back({path[2], path[1]});
        graph[path[1]].push_back({path[2], path[0]});
    }
    
    // set 저장
    for(int i=0; i<gates.size(); i++) gate.insert(gates[i]);
    for(int i=0; i<summits.size(); i++) summit.insert(summits[i]);
    
    // 다익스트라 실행
    vector<int> result = dijkstra(graph, gate, summit);
    
    // 최솟값 찾기
    for(int i=0; i<summits.size(); i++){
        int node = summits[i];
        int weight = result[node];
            
        if(weight < answer[1]){
            answer[0] = node;
            answer[1] = weight;
        }
    }
    
    return answer;
}   
```

* **다익스트라**를 활용하여 풀이
* 일반적인 다익스트라와 달리 경로의 모든 비용을 더하는 것이 아닌, **최댓값만 업데이트** 하는 방식
* 출입구, 산봉우리를 거쳐서 다른 장소로 갈 수 없으므로 **두 장소에 도착하면 탐색을 멈춤**
* 출발지가 여러 개이므로 **우선순위 큐에 시작부터 다 넣으면 결국 최솟값**만 남음

## 3. 에라토스테네스의 체

* 에라토스테네스가 제시한 `n` 보다 작은 수 중 소수를 판별하는 방법
* 체로 수를 걸러내는 듯한 과정을 가져 **에라토스테네스의 체**라고 불림

### 작동 순서
1. $2$부터 $n$까지의 숫자를 나열

2. 아직 지워지지 않은 소수 중 가장 작은 수를 선택 ($2$ 부터 시작)

3. 선택한 수의 배수를 모두 삭제

4. 더 이상 선택할 수가 없을 때까지 반복 ($\sqrt{n}$까지 하면 됨)

5. 삭제되지 않고 남은 수가 $n$보다 작은 모든 소수

### 응용 (약수의 개수)
* 위의 [등산코스 정하기](https://school.programmers.co.kr/learn/courses/30/lessons/118669) 문제에서 원하는 것은 결국 어떤 수 `n`의 **약수의 개수** 임
* 이를 **에라토스테네스의 체**의 방식을 응용하여 계산 가능
* 어떤 수 `i`의 모든 배수에 `count`를 `1`씩 더해주면 모든 수는 자신의 약수 개수를 `count` 하는 것
* 이를 코드로 표현하면 아래와 같음

    ```cpp
    vector<int> count(n+1, 0);
    for(int i=1; i<=n; i++)
        for(int j=i; j<=n; j+=i)
            count[j]++;
    ```

## 4. CH3 개인과제 7번
* Pawn 클래스로 구현하는 과제를 진행하며 드론을 구현해 봄
* 6자유도(6 DOF) 비행체 구현이 도전과제의 내용이었지만, 현실적인 드론의 움직임을 구현해보고자 함

### 드론(쿼드콥터) 유체 역학
* 4개의 프로펠러가 장착된 비행 물체를 **쿼드콥터**라고 함
* 쿼드콥터는 4개의 프로펠러의 회전 방향과 속도를 통해 **양력**과 **추력**을 만들고 몸체를 회전, 이동함
* 양력과 추력의 반대 방향으로 **중력**과 **항력**이 작용하여 최대 속도가 제한됨
* 쿼드콥터는 **이동 방향의 프로펠러의 회전 속도를 낮춰** 기체를 이동 방향 쪽으로 기울여 해당 방향으로 양력을 추력으로 바꿈
* FL/BR 날개는 반시계 방향, FR/BL은 시계 방향으로 프로펠러를 회전시켜 시계 방향, 반시계 방향 **토크를 생성**하고 해당 토크로 Yaw를 변경

### 프로펠러 속도

```cpp
void ADrone::UpdateWingsRPM(float DeltaTime)
{
    float InterpSpeed = 5.0f;
    float Hover = 0.5f;
    float Thrust = ThrustInput * 0.15f;
    float Pitch = MovementInput.Y * 0.2f;
    float Roll = MovementInput.X * 0.2f;
    float Yaw = YawInput * 0.1f;
    float ClampMin =  0.1f;

    if(bIsGrounded)
    {
        Hover = 0.0f;
        Thrust = ThrustInput * 0.15f;
        Pitch = 0.0f;
        Roll = 0.0f;
        Yaw = YawInput * 0.2f;
        InterpSpeed = 0.75f;
        ClampMin = 0.0f;
    }

    TargetWingsRPM[0] = Hover + Thrust - Pitch + Roll + Yaw; // Front Left
    TargetWingsRPM[1] = Hover + Thrust - Pitch - Roll - Yaw; // Front Right
    TargetWingsRPM[2] = Hover + Thrust + Pitch + Roll - Yaw; // Back Left
    TargetWingsRPM[3] = Hover + Thrust + Pitch - Roll + Yaw; // Back Right

    for (int32 i = 0; i < TargetWingsRPM.Num(); i++)
    {
        TargetWingsRPM[i] = FMath::Clamp(TargetWingsRPM[i], ClampMin, 1.0f);
        CurrentWingsRPM[i] = FMath::FInterpTo(CurrentWingsRPM[i], TargetWingsRPM[i], DeltaTime, InterpSpeed);
    }
}
```

* 입력에 따라 가중치를 주어 날개 속도를 조정
* 공중에 떠 있는 경우, `Hover`를 기본값으로 줘 입력 없이도 공중에 떠 있게 유지
* 바닥에 있는 경우에 `Hover`, `Pitch`, `Roll` 을 0으로 바꿔 자동으로 뜨지 않고, 기울어지지 않게 조정

### 기체 회전
```cpp
void ADrone::AirboneRotation(float DeltaTime)
{
    float PitchTorque = (CurrentWingsRPM[0] + CurrentWingsRPM[1]) - (CurrentWingsRPM[2] + CurrentWingsRPM[3]);
    float RollTorque = (CurrentWingsRPM[0] + CurrentWingsRPM[2]) - (CurrentWingsRPM[1] + CurrentWingsRPM[3]);
    float YawTorque = (CurrentWingsRPM[1] + CurrentWingsRPM[2])- (CurrentWingsRPM[0] + CurrentWingsRPM[3]);

    RotationVelocity.Pitch += PitchTorque * RotationPower * DeltaTime;
    RotationVelocity.Roll += RollTorque * RotationPower * DeltaTime;

    if (!FMath::IsNearlyZero(YawInput))
    {
        RotationVelocity.Yaw += YawTorque * RotationPower * YawStrength * DeltaTime;
    }

    RotationVelocity -= RotationVelocity * RotationDamping * DeltaTime;

    FRotator CurrentRotation = GetActorRotation();
    RotationVelocity.Pitch += -CurrentRotation.Pitch * RecoverRotationSpeed * DeltaTime;
    RotationVelocity.Roll += -CurrentRotation.Roll * RecoverRotationSpeed * DeltaTime;
}
```

* 프로펠러의 회전 속도에 따라 각 토크를 구함
* 각 토크와 `RotationPower`를 곱해 기체의 속도를 계산
* `Yaw`의 경우 회전을 위한 토크가 매우 크기 때문에 `YawStrength`를 곱해줘 보다 과장되게 회전

### 기체 이동
```cpp
void ADrone::AirboneMovement(float DeltaTime)
{
    float TotalRPM = 0.0f;
    for (float RPM : CurrentWingsRPM)
    {
        TotalRPM += RPM;
    }

    FVector ThrustDirection = GetActorUpVector();
    FVector ThrustForce = ThrustDirection * TotalRPM * ThrustPower;
    Velocity += ThrustForce * DeltaTime;

    if(!bIsGrounded)
    {
        Velocity -= Velocity * Drag * DeltaTime;
    }
}

void ADrone::GroundedMovement(float DeltaTime)
{
    Velocity.Z = 0.0f;

    FVector Direction = GetActorForwardVector() * MovementInput.Y + GetActorRightVector() * MovementInput.X;

    if (!Direction.IsNearlyZero())
    {
        Direction.Normalize();
        Velocity += Direction * GroundMovePower * DeltaTime;
        Velocity = Velocity.GetClampedToMaxSize(GroundedMaxSpeed);
    }

    else
    {
        Velocity -= Velocity * GroundedFriction * DeltaTime;
    }
}
```

* 공중에 있는 경우에는 드론의 기울임 방향에 따라 이동하게끔 구현
* 바닥에 붙어있는 경우 바닥의 마찰 계수가 적용되기 때문에, 양력을 받을 때 항력은 적용하지 않음
* 바닥에 있는 경우 입력을 통해 벡터의 방향을 구함
* 이후 가속도와 마찰을 계산하여 속도에 더함
