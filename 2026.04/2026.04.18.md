## 2. malloc / free 와 new / delete

### malloc / free

* `C`에서 메모리를 동적으로 할당 및 해제하는데 사용하는 **함수**

* 사용하기 위해서는 `#include <stdlib.h>` 헤더를 추가해야 함

* `malloc()` 함수는 `size`를 인자로 받아 사용 가능한 메모리의 시작 주소를 반환하고 실패할 경우 `NULL`을 반환함

* `void*` 타입으로 반환되기에 수동으로 **캐스팅**을 필요로 함

* **생성자**를 호출하지 않기에, **쓰레기값**이 저장되어 있음

* `calloc()` 함수를 통해 초기화할 수 있지만, `malloc()`보다 약간의 오버헤드가 존재

* `realloc()` 함수를 통해 할당받은 메모리를 확장 가능하지만, 실패 시 **메모리 누수**가 발생하므로 원본을 확장하는 것은 권장하지 않음

* `malloc()` 연속된 메모리를 갖는 `POD(Plain Old Data)` 타입의 데이터만 할당해야 하며, POD 타입이 아닌 데이터를 할당할 경우 **미정의 동작**으로 인해 오류가 발생할 수 있음

* `free()`는 할당된 메모리를 해제하는 **함수**이며, **소멸자**를 호출하지 않음


### 예시 코드

* `malloc / free` 사용법

    ```cpp
    #include <iostream>
    #include <cstdlib>

    int main() {
        // 메모리 할당
        int* pi = (int*)malloc(sizeof(int));
        
        // 할당 여부 확인
        if (pi == nullptr) {
            return 0;
        }

        std::cout << "할당 직후: "<< *pi << std::endl;      // Output: 쓰레기 값
        
        *pi = 10;

        std::cout << "값 수정 후: "<< *pi << std::endl;     // Output: 10

        // 해제 후 초기화
        free(pi);
        pi = nullptr;
        
        return 0;
    }
    ```

* `POD` 타입이 아닌 데이터를 할당한 경우

    ```cpp
    #include <iostream>
    #include <cstdlib>

    // POD 타입이 아닌 클래스
    class Base{
    public:
        int value;

        Base(){
            value = 10;
        }

        ~Base(){
            std::cout << "소멸자 실행" << std::endl;
        }
    };


    int main() {
        Base* Base_malloc = (Base*)malloc(sizeof(Base));
                           
        if (Base_malloc == nullptr) {
            return 0;
        }

        std::cout << "할당 직후: "<< Base_malloc->value << std::endl; // Output: 쓰레기 값
                                                                     // 생성자가 실행되지 않기에 쓰레기 값이 저장되어 있음
        free(Base_malloc);  // 소멸자가 실행되지 않음
        Base_malloc = nullptr;

        return 0;
    }
    ```

* **가상 함수** 사용 시

    ```cpp
    #include <iostream>
    #include <stdlib.h>

    // POD 타입이 아닌 클래스
    class Base{
    public:
        int value;

        Base(){
            value = 10;
        }

        virtual ~Base(){}

        virtual void add(int input){
            value += input;
        }
    };

    // 가상 함수 사용
    class Derived : public Base{
    public:
        void add(int input) override {
            value += input*2;
        }
    };

    int main() {
        Derived* Derived_malloc = (Derived*)malloc(sizeof(Derived));

        if (Derived_malloc == nullptr) {
            return 0;
        }

        //Derived_malloc->add(10);  // 가상 함수 테이블이 없기 때문에 크래시가 발생함

        free(Derived_malloc);
        Derived_malloc = nullptr;

        return 0;
    }
    ```
### new / delete 

* `C++`에서 메모리를 동적으로 할당 및 해제하는데 사용하는 **연산자**

* **배열** 할당 및 해제 시 `new[]`, `delete[]` 를 사용함

* `C++`에서 기본적으로 지원하는 **키워드**이기에, 따로 라이브러리를 `include` 하지 않아도 사용 가능

* `new`는 내부적으로 `malloc()`을 사용해 구현되어 있음

* `new`는 타입을 입력받아, 해당 타입에 알맞는 `size`만큼 메모리를 할당 후 **요청한 타입으로 반환**하며, 실패 시 `std::bad_alloc` 예외를 `throw` 함

* **생성자**를 호출하며, **인자**또한 전달 가능

* 메모리 확장이 **불가능**하며, 확장이 필요한 경우 **재할당** 후 복사를 통해 확장해야 함

* `POD` 타입과 복잡한 관계를 갖는 클래스 또한 할당이 가능함

* `delete`는 할당된 메모리를 해제하는 **연산자**이며, **소멸자**를 호출함

### 예제 코드

* `new / delete 및 new[] / delete[]` 사용법
    ```cpp
    #include <iostream>

    int main(){
        int* pi = new int(100);     // 인자를 넣어 초기화
        int* arr = new int[3];      // 동적 배열 할당

        std::cout << *pi << std::endl;      // Output: 100

        for(int i=0; i<3; i++){
            std::cout << arr[i] << " ";     // Output: 0 0 0
        }                                   // 기본적으로 0 으로 초기화

        // 메모리 해제
        delete pi;
        delete[] arr;       // 배열은 delete[]
    }
    ```

* 복잡한 클래스 할당
    ```cpp
    #include <iostream>

    // POD 타입이 아닌 클래스
    class Base{
    public:
        int value;

        Base(){
            value = 10;
        }

        virtual ~Base(){
            std::cout << "Base 소멸자" << std::endl;
        }

        virtual void add(int input){
            value += input;
        }
    };

    // 가상 함수 사용
    class Derived : public Base{
    public:
        virtual ~Derived(){
            std::cout << "Derived 소멸자" << std::endl;
        }

        void add(int input) override {
            value += input*2;
        }
    };

    int main() {
        Base* Base_new = new Base();
        Derived* Derived_new = new Derived();

        std::cout << Base_new->value << std::endl;      // Output: 10
                                                        // 생성자가 정상적으로 호출
        
        Derived_new->add(10);
        std::cout << Derived_new->value << std::endl;   // Output: 30
                                                        // 가상 함수도 정상적으로 실행됨
        delete Base_new;
        delete Derived_new;
        
        return 0;
    }
    /*Output:       소멸자도 정상적으로 호출
    Base 소멸자
    Derived 소멸자
    Base 소멸자
    */
    ```

### 차이점

* `malloc / free` 는 함수이며, `new / delete` 는 연산자

* `malloc / free` 는 생성자 및 소멸자를 호출하지 않지만, `new / delete` 는 호 함

* `malloc`은 캐스팅을 필요로 하지만, `new`는 해당 타입의 포인터를 반환함

* `malloc`은 할당 실패 시 `NULL`을 반환하며, `new`는 `std::bad_alloc` 예외를 `throw` 함

* `realloc`을 통해 확장이 가능하지만, `new` 새롭게 할당 후 복사해야 함

* `malloc`은 배열 할당 시 개발자가 직접 size를 계산해야 하지만 , `new[]`를 통해 안전하게 배열을 할당할 수 있음

* `malloc / free` 는 라이브러리 함수이기에 재정의가 불가능하지만, `new / delete` 는 연산자 오버로딩을 통한 재정의가 가능함
