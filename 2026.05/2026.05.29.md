## 2. 프로그래머스 문제 풀이

### [디펜스 게임](https://school.programmers.co.kr/learn/courses/30/lessons/142085)

```cpp
#include <string>
#include <vector>
#include <queue>

using namespace std;

int solution(int n, int k, vector<int> enemy) {
    int answer = 0;
    priority_queue<int> pq;
    for(int i=0; i<enemy.size(); i++){
        n -= enemy[i];
        pq.push(enemy[i]);
        
        while(!pq.empty() && n < 0 && k > 0){
            n += pq.top();
            pq.pop();
            k--;
        }
        if(n < 0) return i;
    }
    return enemy.size();
}
```

* **그리디** 유형의 문제
* 모든 적을 **무적권을 사용하지 않는다**고 생각하고 **우선순위 큐**에 적의 수를 저장
* `n`이 **음수**가 되었을 때, **가장 높은 적의 수**에 무적권을 사용
* 무적권을 **모두 사용**해도 `n`이 음수면 전 라운드까지 막을 수 있는 것

### [과제 진행하기](https://school.programmers.co.kr/learn/courses/30/lessons/176962)

```cpp
#include <string>
#include <vector>
#include <stack>
#include <algorithm>
#include <unordered_map>
using namespace std;

vector<string> solution(vector<vector<string>> plans) {
    vector<string> answer;
    stack<int> s;
    unordered_map<int, string> index;
    vector<vector<int>> intPlans;
    
    sort(plans.begin(), plans.end(), [](const vector<string>& A, const vector<string>& B){
        return A[1] < B[1];
    });
    
    for(int i=0; i<plans.size(); i++){
        index[i] = plans[i][0];
        int time = 0;
        time += stoi(plans[i][1].substr(0, 2)) * 60;
        time += stoi(plans[i][1].substr(3, 2));
        intPlans.push_back({i, time, stoi(plans[i][2])});
    }
    
    int current = 0;
    int nextCur = current + 1;

    for(int i=intPlans[0][1]; ; i++){
        if(--intPlans[current][2] == 0) {
            answer.push_back(index[current]);
            if(!s.empty()){
                current = s.top();
                s.pop();
            }
        }
        
        if(nextCur < intPlans.size()){
            vector<int> next = intPlans[nextCur];
            if(next[1] == i){
                if(intPlans[current][2] != 0) s.push(current);
                current = next[0];
                nextCur++;
            }  
        }
        else if(s.empty() && answer.size() == intPlans.size())
            break;        
    }    
    return answer;
}
```

* **구현** 문제
* **스케쥴링**을 구현하는 유형의 문제
* 시간의 범위가 매우 좁기 때문에 `(0 ~ 24*60 - 1)` 시뮬레이션으로 해도 충분함
* 시간의 범위가 넓을 경우 **시간차를 계산**하는 방식으로 수정해야 함
