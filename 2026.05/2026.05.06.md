### [최댓값과 최솟값](https://school.programmers.co.kr/learn/courses/30/lessons/12939)

```cpp
#include <string>
#include <vector>
#include <sstream>
#include <climits>
using namespace std;


string solution(string s) {
    stringstream ss(s);
    string temp;
    int MIN = INT_MAX, MAX = INT_MIN; 
    while(getline(ss, temp, ' ')){
        MIN = min(MIN, stoi(temp));
        MAX = max(MAX, stoi(temp));
    }
        
    return to_string(MIN) + ' ' + to_string(MAX);
}
```

* **문자열**을 문제
* 공백을 기준으로 문자열을 분리 후, `stoi()` 함수로 최대, 최소를 판별

### [지게차와 크레인](https://school.programmers.co.kr/learn/courses/30/lessons/388353)

```cpp
#include <string>
#include <vector>
#include <queue>

using namespace std;

void BFS(vector<string>& storage, const string& request, int n, int m){
    queue<pair<int, int>> q;
    queue<pair<int, int>> deleteQueue;
    int dx[4] = {-1, 1, 0, 0};
    int dy[4] = {0, 0, -1, 1};
    vector<vector<bool>> visit(n, vector<bool>(m, false));
    q.push({0, 0});
    visit[0][0] = true;
    
    while(!q.empty()){
        int y = q.front().first;
        int x = q.front().second;
        q.pop();
        
        if(storage[y][x] != ' ') continue;
        for(int i=0; i<4; i++){
            int nextY = y + dy[i];
            int nextX = x + dx[i];
            
            if(nextY<0 || nextY>=n || nextX<0 || nextX>=m || visit[nextY][nextX]) 
                continue;

            q.push({nextY, nextX});
            visit[nextY][nextX] = true;
            if(storage[nextY][nextX] == request[0]) deleteQueue.push({nextY, nextX});
        }
    }
    
    while(!deleteQueue.empty()){
        int y = deleteQueue.front().first;
        int x = deleteQueue.front().second;
        deleteQueue.pop();
        storage[y][x] = ' ';
    }
}

int solution(vector<string> storage, vector<string> requests) {
    int answer = 0;
    
    string temp(storage[0].length()+2, ' ');
    storage.push_back(temp); storage.push_back(temp);
    
    for(int i=storage.size()-2; i>0; i--){
        storage[i-1] = " " + storage[i-1] + " ";
        storage[i] = storage[i-1];
    }
    storage[0] = temp;
    int n = storage.size(), m = storage[0].size();
    
    for(const string& request : requests){
        if(request.length() == 1) BFS(storage, request, n, m);
        
        else{
            for(int i=0; i<n; i++)
                for(int j=0; j<m; j++)
                    if(storage[i][j] == request[0]) 
                        storage[i][j] = ' ';
        }
    }

    for(int i=0; i<n; i++)
        for(int j=0; j<m; j++)
            if(storage[i][j] != ' ') 
                answer++;

    return answer;
}
```

* **BFS**를 활용한 탐색 문제
* `storage`를 확장하여 지게차로 도달 가능한 지점을 **BFS**로 탐색
* 크레인을 사용하는 경우에는 **완전 탐색**으로 지움

## 3. std::sort()

* `<algorithm>` 에 정의된 **정렬** 함수

* 함수 내부에서 데이터의 상태에 따라서 **퀵 정렬**, **힙 정렬**, **삽입 정렬** 중 선택하는 `인트로 정렬(Introsort)`을 사용

* 입력 인자로 시작점의 **이터레이터**`(first)`와 종료 지점의 **이터레이터**`(last)`를 받으며, `[first, last)` 범위를 **오름차순**으로 정렬

* 세 번째 인자로 **비교 함수**를 작성해 원하는 방식으로 정렬 가능

* `std::sort()`는 평균적으로 $O(NLogN)$의 시간 복잡도를 가짐

* `first`와 `last`는 `vector`와 같이 **임의 접근 반복자**이어야 함

*  **순차 접근 반복자**를 사용하는 컨테이너는 `std::list`, `std::forward_list`와 `연관 컨테이너`가 있음
* `std::list`, `std::forward_list`의 경우 멤버로 `std::sort()`가 존재하지만 $O(N)$의 시간 복잡도를 가짐

* `연관 컨테이너`는 기본적으로 정렬을 지원하거나, 정렬이 불가능한 형태이므로 `sort()`가 불가능
