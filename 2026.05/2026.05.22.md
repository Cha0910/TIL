### [모의고사](https://school.programmers.co.kr/learn/courses/30/lessons/42840)

```cpp
#include <string>
#include <vector>
#include <algorithm>
using namespace std;

vector<int> solution(vector<int> answers) {
    vector<int> answer;
    vector<vector<int>> arr = {{1, 2, 3, 4, 5},
                               {2, 1, 2, 3, 2, 4, 2, 5},
                               {3, 3, 1, 1, 2, 2, 4, 4, 5, 5}};
    vector<int> count (3, 0);
    for(int i=0; i<answers.size(); i++){
        if(arr[0][i%5] == answers[i]) count[0]++;
        if(arr[1][i%8] == answers[i]) count[1]++;
        if(arr[2][i%10] == answers[i]) count[2]++;
    }
    int MAX = *max_element(count.begin(), count.end());
    for(int i=0; i<count.size(); i++)
        if(MAX == count[i]) answer.push_back(i+1);
    return answer;
}
```

* **완전 탐색** 문제
* 나머지 연산으로 정답의 개수를 저장
* 가장 많은 정답을 맞춘 사람을 answer에 저장

### [로또의 최고 순위와 최저 순위](https://school.programmers.co.kr/learn/courses/30/lessons/77484)

```cpp
#include <string>
#include <vector>
#include <set>
using namespace std;

vector<int> solution(vector<int> lottos, vector<int> win_nums) {
    vector<int> answer(2, 0);
    vector<int> rank {6, 6, 5, 4, 3, 2, 1};
    set<int> win(win_nums.begin(), win_nums.end());
    for(int i=0; i<lottos.size(); i++){
        if(lottos[i] == 0 || win.count(lottos[i])) answer[0]++;
        if(win.count(lottos[i])) answer[1]++;
    }
    return {rank[answer[0]], rank[answer[1]]};
}
```

* **구현** 문제
* `set`을 통해 당첨 번호와 로또 번호를 비교
* 배열 인덱싱으로 맞춘 개수에 따른 순위를 출력
