## 2. 프로그래머스 문제 풀이

### [기사단원의 무기](https://school.programmers.co.kr/learn/courses/30/lessons/136798)

```cpp
#include <string>
#include <vector>

using namespace std;

int solution(int number, int limit, int power) {
    int answer = 0;
    vector<int> count(number+1, 0);
    for(int i=1; i<count.size(); i++)
        for(int j=i; j<count.size(); j+=i)
            count[j]++;
        
    for(int i=1; i<count.size(); i++)
        answer += count[i] > limit ? power : count[i];

    return answer;
}
```

* **수학** 문제
* **에라토스테네스의 체**를 활용하여 약수를 카운팅 `O(NlogN)`
* number의 최대값이 10만이므로, 제곱근 방식으로 해도 충분함

### [덧칠하기](https://school.programmers.co.kr/learn/courses/30/lessons/161989)

```cpp
#include <string>
#include <vector>

using namespace std;

int solution(int n, int m, vector<int> section) {
    int answer = 0;
    vector<bool> wall(n, true);
    for(int i=0; i<section.size(); i++) wall[section[i]-1] = false;
    
    for(int i=0; i<n; i++){
        if(!wall[i]){
            fill(wall.begin()+i, min(wall.begin()+i+m, wall.end()), true);
            answer++;
        } 
    }
    return answer;
}
```

* **그리디** 유형의 문제
* 색칠되지 않은 벽을 만나면 `m` 길이 만큼 뒤의 벽까지 칠
