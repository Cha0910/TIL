## 2. 프로그래머스 문제 풀이

### [도넛과 막대 그래프](https://school.programmers.co.kr/learn/courses/30/lessons/258711)

```cpp
#include <string>
#include <vector>
#include <queue>

using namespace std;

int BFS(const vector<vector<int>>& graph, int start){
    int num_edges = 0, num_nodes = 0;
    vector<bool> visit(graph.size(), false);
    queue<int> q;
    
    q.push(start);
    visit[start] = true;
    num_nodes++;
    num_edges += graph[start].size();
    
    while(!q.empty()){
        int current = q.front();
        q.pop();
        
        for(int i=0; i<graph[current].size(); i++){
            int next = graph[current][i];
            if(visit[next]) continue;
            q.push(next);
            visit[next] = true;
            num_nodes++;
            num_edges += graph[next].size();
        }
    }
    if(num_nodes == num_edges) return 1;
    else if(num_nodes == num_edges+1) return 2;
    else return 3;
}

vector<int> solution(vector<vector<int>> edges) {
    vector<int> answer(4, 0);
    int max = 0;
    for(const vector<int>& edge : edges){
        max = max > edge[0] ? max : edge[0];
        max = max > edge[1] ? max : edge[1];
    }
    max++;
    vector<int> count(max, 0);
    vector<vector<int>> graph(max);
    
    for(const vector<int>& edge : edges){
        graph[edge[0]].push_back(edge[1]);
        count[edge[1]]++;
    }
    
    int root;
    for(int i=1; i<max; i++){
        if(count[i] == 0 && graph[i].size() > 1){
            root = i;
            answer[0] = root;
            break;
        } 
    }
    
    for(int i=0; i<graph[root].size(); i++){
        int next = graph[root][i];
        answer[BFS(graph, next)]++;
    }
    
    return answer;
}
```

* **BFS**를 활용해 풀이
* 몇 번 노드가 추가된 노드인지 확인하는 것이 가장 중요한 문제
* 추가된 노드의 조건은 **들어오는 간선이 없고, 나가는 간선이 2개 이상**이어야 함 (기존의 그래프가 2개 이상이므로)
* **추가된 노드에서 연결된 노드들을 탐색**하여 노드와 간선의 개수로 그래프의 모양을 알 수 있음

### 2606번: 바이러스
- **Tier**: N/A
- **시간 제한**: 1 초
- **메모리 제한**: 128 MB

#### 문제 설명
신종 바이러스인 웜 바이러스는 네트워크를 통해 전파된다. 한 컴퓨터가 웜 바이러스에 걸리면 그 컴퓨터와 네트워크 상에서 연결되어 있는 모든 컴퓨터는 웜 바이러스에 걸리게 된다.
예를 들어 7대의 컴퓨터가 <그림 1>과 같이 네트워크 상에서 연결되어 있다고 하자. 1번 컴퓨터가 웜 바이러스에 걸리면 웜 바이러스는 2번과 5번 컴퓨터를 거쳐 3번과 6번 컴퓨터까지 전파되어 2, 3, 5, 6 네 대의 컴퓨터는 웜 바이러스에 걸리게 된다. 하지만 4번과 7번 컴퓨터는 1번 컴퓨터와 네트워크상에서 연결되어 있지 않기 때문에 영향을 받지 않는다.
어느 날 1번 컴퓨터가 웜 바이러스에 걸렸다. 컴퓨터의 수와 네트워크 상에서 서로 연결되어 있는 정보가 주어질 때, 1번 컴퓨터를 통해 웜 바이러스에 걸리게 되는 컴퓨터의 수를 출력하는 프로그램을 작성하시오.

#### 입력
첫째 줄에는 컴퓨터의 수가 주어진다. 컴퓨터의 수는 100 이하인 양의 정수이고 각 컴퓨터에는 1번 부터 차례대로 번호가 매겨진다. 둘째 줄에는 네트워크 상에서 직접 연결되어 있는 컴퓨터 쌍의 수가 주어진다. 이어서 그 수만큼 한 줄에 한 쌍씩 네트워크 상에서 직접 연결되어 있는 컴퓨터의 번호 쌍이 주어진다.

#### 출력
1번 컴퓨터가 웜 바이러스에 걸렸을 때, 1번 컴퓨터를 통해 웜 바이러스에 걸리게 되는 컴퓨터의 수를 첫째 줄에 출력한다.

##### 예제 입력 1
```
7
6
1 2
2 3
1 5
5 2
5 6
4 7
```
#### 예제 출력 1
```
4
```

#### 정답 코드
```cpp
#include <iostream>
#include <vector>

using namespace std;

int DFS(const vector<vector<int>>& graph, vector<bool>& visit, int curr){
    if(visit[curr]) return 0;
    int result = 0;
    visit[curr] = true;
    result++;
    
    for(const int next : graph[curr])
        result += DFS(graph, visit, next);

    return result;
}

int main() {
    int N, M;
    cin >> N >> M;
    vector<vector<int>> graph(N+1);
    vector<bool> visit(graph.size(), false);
    for(int i=0; i<M; i++){
        int A, B;
        cin >> A >> B;
        graph[A].push_back(B);
        graph[B].push_back(A);
    }

    cout<< DFS(graph, visit, 1)-1 << " ";
    
    return 0;
}
```

* **DFS**를 활용해 풀이
* 노드를 방문할 때마다 result 1씩 증가시켜 반환
* 1번 컴퓨터는 감염시킨 것이 아니므로 결과에서 1을 뺌

### 2644번: 촌수계산
- **Tier**: N/A
- **시간 제한**: 1 초
- **메모리 제한**: 128 MB

#### 문제 설명
우리 나라는 가족 혹은 친척들 사이의 관계를 촌수라는 단위로 표현하는 독특한 문화를 가지고 있다. 이러한 촌수는 다음과 같은 방식으로 계산된다. 기본적으로 부모와 자식 사이를 1촌으로 정의하고 이로부터 사람들 간의 촌수를 계산한다. 예를 들면 나와 아버지, 아버지와 할아버지는 각각 1촌으로 나와 할아버지는 2촌이 되고, 아버지 형제들과 할아버지는 1촌, 나와 아버지 형제들과는 3촌이 된다.
여러 사람들에 대한 부모 자식들 간의 관계가 주어졌을 때, 주어진 두 사람의 촌수를 계산하는 프로그램을 작성하시오.

#### 입력
사람들은 1, 2, 3, …, n (1 ≤ n ≤ 100)의 연속된 번호로 각각 표시된다. 입력 파일의 첫째 줄에는 전체 사람의 수 n이 주어지고, 둘째 줄에는 촌수를 계산해야 하는 서로 다른 두 사람의 번호가 주어진다. 그리고 셋째 줄에는 부모 자식들 간의 관계의 개수 m이 주어진다. 넷째 줄부터는 부모 자식간의 관계를 나타내는 두 번호 x,y가 각 줄에 나온다. 이때 앞에 나오는 번호 x는 뒤에 나오는 정수 y의 부모 번호를 나타낸다.
각 사람의 부모는 최대 한 명만 주어진다.

#### 출력
입력에서 요구한 두 사람의 촌수를 나타내는 정수를 출력한다. 어떤 경우에는 두 사람의 친척 관계가 전혀 없어 촌수를 계산할 수 없을 때가 있다. 이때에는 -1을 출력해야 한다.

#### 예제 입력 1
```
9
7 3
7
1 2
1 3
2 7
2 8
2 9
4 5
4 6
```
#### 예제 출력 1
```
3
```

#### 예제 입력 2
```
9
8 6
7
1 2
1 3
2 7
2 8
2 9
4 5
4 6
```
#### 예제 출력 2
```
-1
```

#### 정답 코드
```cpp
#include <iostream>
#include <vector>
#include <queue>
using namespace std;

void BFS(const vector<vector<int>>& graph, vector<int>& visit, int start){
    queue<int> q;
    q.push(start);
    visit[start] = 0;

    while (!q.empty()){
        int curr = q.front();
        q.pop();

        for (int next : graph[curr]){
            if(visit[next] != -1) continue;
            
            visit[next] = visit[curr] + 1;
            q.push(next);
        }
    } 
}

int main() {
    int n, start, target, m;
    cin >> n >> start >> target >> m;
    vector<vector<int>> graph(n+1);
    vector<int> visit(graph.size(), -1);
    
    for(int i=0; i<m; i++){
        int A, B;
        cin >> A >> B;
        graph[A].push_back(B);
        graph[B].push_back(A);
    }

    BFS(graph, visit, start);
    cout << visit[target];
    
    return 0;
}
```

* **BFS**를 활용하여 풀이
* 촌수는 **각 노드의 거리**와 같음

## 4. C++과 언리얼의 RTTI (Run Time Type Information)

### C++의 RTTI

* C++ 에서 RTTI를 지원하는 제공은 `typeid()`, `dynamic_cast<T>()` 두 가지가 존재

* `typeid()`: 타입의 **기본적인 정보**(type_info / 이름, 해시코드 등)를 반환하는 연산자

* `dynamic_cast<T>()`: 안전하게 **다운 캐스팅**할 때 사용하는 연산자

* `dynamic_cast`를 통해 다운 캐스팅 하는 경우, **vtable을 반드시 필요**로 하며 캐스팅 검증을 위해 **vtable의 상속 관계를 탐색 및 비교**하므로 매우 비싼 연산임

### 언리얼의 RTTI

* C++의 RTTI는 제공하는 정보가 매우 적고, 성능상의 이슈가 있어 **사용하지 않음**

* 언리얼은 **리플랙션 시스템**과 **RTTI**를 자체적으로 구현

* 언리얼의 `UCLASS()` 매크로가 등록된 모든 클래스는 **메타데이터**를 저장하는 `UClass`를 생성하며, 이후 생성된 모든 오브젝트는 해당 객체의 주소를 갖는 `UClass*`를 갖고있음

* `UClass`는 **싱글톤** 객체이므로 반드시 하나만 존재하며 **포인터 비교**만으로 타입 검사가 가능

* 언리얼에서는 `IsA()`, `Cast<T>()`를 통해 **다운 캐스팅**을 지원

* `Cast<T>()`는 AActor, APawn 등과 같은 자주 쓰이는 핵심 오브젝트에 `CastFlags`를 부여해 **비트연산으로 매우 빠르게 캐스팅**을 실행

* `CastFlags`가 없는 경우 `IsA()`를 통하여 다운 캐스팅함
