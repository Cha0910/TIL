## 2. 프로그래머스 문제 풀이

### [올바른 괄호](https://school.programmers.co.kr/learn/courses/30/lessons/12909)

```cpp
#include <string>
#include <iostream>

using namespace std;

bool solution(string s)
{
    int count = 0;
    for(const char c : s){
        if(c == '(') count++;
        else if(--count < 0) return false;
    }

    return count == 0;
}
```

* 간단한 **스택** 문제
* 스택 안에 들어가는 문자가 `(` 밖에 없으므로 굳이 스택을 직접 구현하는 게 아닌, **몇 개가 들어갔는지만 표시**하면 됨

### [숫자의 표현](https://school.programmers.co.kr/learn/courses/30/lessons/12924)

```cpp
#include <string>
#include <vector>

using namespace std;

int solution(int n) {
    int answer = 0;
    int left = 1, right = 1, sum = 1;
    
    while(left <= right){
        if(sum < n) sum += ++right;
        else if (sum > n) sum -= left++;
        else{
            answer++;
            sum += ++right;
        }
    }
    
    return answer;
}
```

* **슬라이딩 윈도우** 문제
* 숫자 조합을 투 포인터로 바꿔가면서 총합이 `n`과 같은 경우 `answer`를 증가

### [조이스틱](https://school.programmers.co.kr/learn/courses/30/lessons/42860)

```cpp
#include <string>
#include <vector>
using namespace std;

int getNext(const string& name, int start){
    for(int i=start; i<name.length(); i++)
        if(name[i] != 'A') return i;
    return name.length();
}

int solution(string name) {
    int answer = 0;
    vector<int> alpha(26);
    for(int i=0; i<alpha.size(); i++) alpha[i] = min(i, 26-i);
    
    int len = name.length();
    int move = len - 1;    
    for(int i=0; i<name.length(); i++){
        int next = getNext(name, i+1);
        move = min(move, (i * 2) + len - next);
        move = min(move, (len - next) * 2 + i);
        answer += alpha[name[i] - 'A'];
    }
    
    return answer + move;
}
```

* **완전 탐색** 방식으로 풀이
* 좋은 방식은 아니지만, 문제 유형이 **그리디**로 분류되어 있어 **가장 가까운 목표로 이동**하는 것이 최선의 경우인 줄 알고 문제를 풀었지만, 오답이 나옴
* 문제의 핵심은 **방향 전환은 최대 한 번**만 하는 것이 최적이라는 것
* 즉, 시작하자마자 왼쪽으로 이동하고 어떤 지점에서 오른쪽에서 꺾은 경우, 시작하자마자 오른쪽으로 이동하고 어떤 지점에서 왼쪽으로 꺾은 경우 중 낮은 값이 최선의 경우
* 이때 어떤 지점은 `i`번째 지점 뒤에 있는 문자 중 **처음으로 A가 아닌 문자**의 인덱스
