### [거리두기 확인하기](https://school.programmers.co.kr/learn/courses/30/lessons/81302)

```cpp
#include <string>
#include <vector>
#include <queue>

using namespace std;
int BFS(const vector<string>& place){
    queue<pair<int, int>> q;
    
    int dx[4] = {-1, 1, 0, 0};
    int dy[4] = {0, 0, -1, 1};
    
    for(int i=0; i<place.size(); i++){
        for(int j=0; j<place[0].size(); j++){
            vector<vector<int>> visit(place.size(), vector<int>(place[0].size(), -1));
            if(place[i][j] == 'P' && visit[i][j] == -1){
                q.push({i, j});
                visit[i][j] = 0;
            }
            
            while(!q.empty()){
                int y = q.front().first;
                int x = q.front().second;
                q.pop();
                
                for(int k=0; k<4; k++){
                    int ny = y + dy[k];
                    int nx = x + dx[k];
                    if(ny<0 || ny>=place.size() || nx<0 || nx>=place[0].size() || 
                      place[ny][nx] == 'X' || visit[ny][nx] != -1) continue;
                    
                    q.push({ny, nx});
                    visit[ny][nx] = visit[y][x] + 1;
                    if(place[ny][nx] == 'P' && visit[ny][nx] <= 2) return 0;
                }
            }
        }
    }
    return 1;
}
vector<int> solution(vector<vector<string>> places) {
    vector<int> answer;
    for(const vector<string>& place : places)
        answer.push_back(BFS(place));
    return answer;
}
```

* **BFS**를 활용하는 문제
* **모든 P**에 대해 BFS를 실행해 모든 맨해튼 거리를 확인
* 맨해튼 거리가 하나라도 `2` 이하면 `0`을 반환

### [문자열 압축](https://school.programmers.co.kr/learn/courses/30/lessons/60057)

```cpp
#include <string>
#include <vector>

using namespace std;

int solution(string s) {
    int answer = s.length();
    
    for(int i=1; i<=s.length(); i++){
        vector<pair<string, int>> subs;
        string str = "";
        int cur = 0;
        
        while(cur < s.length()){
            string sub = s.substr(cur, i);
            if(!subs.empty() && sub == subs.back().first) subs.back().second++;
            else subs.push_back({sub, 1});
            cur += i;
        }
        
        for(const auto& [sub, count] : subs){
            if(count > 1) str += to_string(count);
            str += sub;
        }
        answer = min(answer, (int)str.length());
    }
    return answer;
}
```

* **문자열**, **완전 탐색** 문제
* 문자열을 `i` 길이로 자른 후 각 자른 문자열을 `subs`에 저장
* 현재 자른 문자열이 가장 최근에 자른 문자열과 동일하면 **카운트를 증가**
* 압축이 모두 끝난 후 `subs`를 순회하며 압축 결과의 길이를 계산

## 3. 백준 문제풀이

### BOJ_2852 NBA 농구

#### 문제 설명
동혁이는 NBA 농구 경기를 즐겨 본다. 동혁이는 골이 들어갈 때 마다 골이 들어간 시간과 팀을 적는 이상한 취미를 가지고 있다.
농구 경기는 정확히 48분동안 진행된다. 각 팀이 몇 분동안 이기고 있었는지 출력하는 프로그램을 작성하시오.

#### 입력
첫째 줄에 골이 들어간 횟수 N(1<=N<=100)이 주어진다. 둘째 줄부터 N개의 줄에 득점 정보가 주어진다. 득점 정보는 득점한 팀의 번호와 득점한 시간으로 이루어져 있다. 팀 번호는 1 또는 2이다. 득점한 시간은 MM:SS(분:초) 형식이며, 분과 초가 한자리 일 경우 첫째자리가 0이다. 분은 0보다 크거나 같고, 47보다 작거나 같으며, 초는 0보다 크거나 같고, 59보다 작거나 같다. 득점 시간이 겹치는 경우는 없다.

#### 출력
첫째 줄에 1번 팀이 이기고 있던 시간, 둘째 줄에 2번 팀이 이기고 있던 시간을 출력한다. 시간은 입력과 같은 형식(MM:SS)으로 출력한다.

#### 코드

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <sstream>

using namespace std;

int strToInt(const string& str){
    vector<string> result;
    stringstream ss(str);
    string temp;
    while(getline(ss, temp, ':'))
        result.push_back(temp);
    return stoi(result[0])*60 + stoi(result[1]);
}

string intTostr(int time){
    string m = to_string(time/60);
    string s = to_string(time%60);
    m = m.length() == 1 ? '0' + m : m;
    s = s.length() == 1 ? '0' + s : s;
    return m + ':' + s;
}

int main() {
    int N, team, prev = 0;
    vector<int> score(2, 0), winTime(2, 0);
    string time;
    cin >> N;

    for(int i=0; i<N; i++){
        cin >> team >> time;
        int intTime = strToInt(time);
        
        if(prev != 0 && score[0] != score[1]){
            int target = score[0] > score[1] ? 0 : 1;
            winTime[target] += intTime - prev;
        }
        
        score[team-1]++; 
        prev = intTime;
    }

    if(score[0] != score[1]){
        int target = score[0] > score[1] ? 0 : 1;
        winTime[target] += 48*60 - prev;
    }
    
    cout << intTostr(winTime[0]) << "\n" << intTostr(winTime[1]);
    
    return 0;
}
```

* **문자열**, **시뮬레이션** 문제
* 경기의 승패 결과는 득점마다 변경 될 수 있으므로, **득점이 된 경우에만 시간을 계산**해 줌
* 시간의 범위가 매우 좁기에, **우선순위 큐**를 사용해 1초 단위로 계산해도 충분히 풀 수 있음

## 4. USTRUCT와 UCLASS

## USTRUCT

* **USTRUCT**는 멤버 변수를 조직화, 조작할 수 있도록 도와주는 **데이터 구조**

* **스택 영역**에서 할당할 수 있으며, 기본적으로 **값 복사 (Pass By Value)** 가 발생

* **USTRUCT** 간에는 상속이 가능하지만, **UObject를 상속받을 수 없음**

* **USTRUCT** 자체는 GC의 관리 대상이 아니며, 내부의 **UPROPERTY** 멤버 변수는 GC가 관리

* **USTRUCT** 자체는 리플리케이션 대상이 아니며, 내부의 **UPROPERTY** 멤버 변수는 리플리케이션 대상

* 내부에 함수를 선언이나 정의할 수 있지만, **UFUNCTION 매크로는 사용 불가능**해 권장하지 않음

* **BlueprintType**으로 지정 시 블루프린트와 연동을 위해 **자동으로 Make, Break 함수를 생성**

* 단순히 데이터를 묶는 데 주로 사용하므로, **일반적으로 가볍고 빠름**

* 보통 접두어 **F**를 붙여 사용

* 예시 코드
    ```cpp
    USTRUCT([Specifier, Specifier, ...])
        struct FStructName
        {
            GENERATED_BODY()
        };
    ```

### UCLASS

* **UObject**를 상속받는 클래스에 붙여 리플랙션에 등록하는 매크로

* **힙 영역**에서만 할당할 수 있으며, 기본적으로 **참조 전달 (Pass By Reference)** 이 발생

* **UCLASS** 매크로가 붙은 **UObject 타입을 상속할 수 있음**

* **UCLASS** 자체와 내부의 **UPROPERTY** 멤버 변수는 GC 및 리플리케이션 대상

* **UFUNCION** 매크로를 사용해 에디터 연동, 네트워크 등의 기능을 활용할 수 있음

* 다양한 기능을 지원하므로 **USTRUCT에 비해 무겁고 오버헤드가 존재**

* 어떤 클래스를 상속받았는지에 따라 **U**, **A** 등의 접두어를 사용

* 예시 코드
    ```cpp
    #pragma once
 
	#include 'Object.h'
	#include 'MyObject.generated.h'
	
	UCLASS()
	class MYPROJECT_API UMyObject : public UObject
	{
		GENERATED_BODY()
 
	};
    ```

### 차이점

|항목|USTRUCT|UCLASS|
|---|---|---|
|메모리|주로 **스택** 영역|반드시 **힙** 영역|
|복사|**값 복사**(Pass By Value)|**참조 전달**(Pass By Reference)|
|상속|**USTRUCT** 만 가능|**UObject 기반**의 UCLASS만 가능|
|리플랙션|**UPROPERTY()** 만 사용 가능|**UPROPERTY()**, **UFUNCTION()** 사용 가능|
|GC/리플리케이션|내부의 **UPROPERTY 변수만** 관리|**클래스 자체**와 내부의 **UPROPERTY 변수** 모두 관리|
|오버헤드|상대적으로 가볍고 더 빠름|상대적으로 **오버헤드** 존재|
|접두어|보통 **F**만 사용|부모에 따라 보통 **U**, **A** 사용|
|주 용도|단순 데이터 보관 및 전달|객체 관리, 복잡한 로직 구현|
